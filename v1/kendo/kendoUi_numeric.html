# NumericTextBox

input42 - 높이가 42px/대타이틀 옆 상단 검색 기능 input style
input35 - 일반적으로 쓰이는 input style
input35 / Number text - 개월, 금액 일때 옆에 label text가 들어갈 경우 사용하는 스타일
(개월, 금액의 경우 오른쪽 정렬을 원칙으로 하며, 그 외의 입력사항에 대해서는 왼쪽정렬을 기준으로 한다.)
input35 / td inner - 스플리터 안 테이블 내 td에 들어가는 input style
