기본점검

제목 각 페이지별 고유한 제목이 설정되어 있는가 브라우저 탭, SR 읽기 순서 고려
의미 없는 기호로 제목을 장식하고 있지 않은가 예: ☆공지사항☆ → X
콘텐츠 구조 콘텐츠 영역별로 제목이나 구분이 되어 있는가 시각적+논리적 구조를 모두 고려
텍스트 표현 기호·약어 사용 시 의미가 명확히 정의되어 있는가 ↓ 할인, P: 포인트 적립 등
색, 위치, 방향, 크기만으로 정보를 전달하지 않는가 예: “왼쪽 버튼 클릭” → X
링크 표현 “여기 클릭”과 같은 추상적인 텍스트는 없는가 “지원서 작성 페이지로 이동” → O
이미지/아이콘 텍스트 없는 이미지에 대체 텍스트를 정의했는가 아이콘, 약도, 차트 등
멀티미디어 영상/음성 콘텐츠에 자막, 해설, 텍스트 대체가 있는가 시각·청각장애 사용자 고려
자동재생 자동으로 재생되는 영상·음향이 있으면 3초 이내인가 또는 수동 조작 기능 제공
깜빡임 빠른 깜빡임 콘텐츠(1초당 3~50회)는 없는가 시각 민감성 사용자 발작 위험 방지
표 구조 의미 있는<table>에 제목(<caption>)을 정의했는가 데이터 표에 적용
마우스 기반 기능 마우스오버 외 다른 방식(키보드, 터치)도 동작 가능한가 메뉴 열기, 툴팁 등
자동 롤링 배너 등 자동 전환 콘텐츠에 정지/탐색 기능이 있는가 정지버튼, 포커스 시 멈춤 등
팝업 구성 의도치 않게 팝업이 자동 실행되진 않는가 자동 알림, 광고 등 제한
입력폼 label 없는 입력 요소에 안내 문구(title 등)가 있는가 title 또는 aria-label 등
select 요소는 submit 없이 페이지 이동하지 않는가 분류형 select는 확인 버튼 필요
자동 포커스 이동 기능을 요구하지 않는가 전화번호 입력 후 자동 이동 등

사용자 요구에 따른 실행 입력요소에 값을 선택, 입력하거나 변경했을 때 자동적으로 초점이 이동하거나 팝업이 뜨거나 페이지 이동이 되지는 않는가?
팝업이 열리는 기능의 링크 또는 버튼에 마우스오버 했을 때 새 창이 열림을 안내하는 툴팁(tooltip=말풍선)이 나오는가?
